# Freefish 咸鱼社区

## 环境准备

安装 `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## 提供的脚本

### 启动项目

```bash
npm start
```

### 构建项目

```bash
npm run build
```

### 在线体验

> http://freefish.love

### 项目主要业务及实现的功能

本项目类似一个简版的掘金这样的技术社区，实现了多个用户注册，登录，发帖，评论，关注，搜索等功能。

### 前端技术栈

Vue

Vuex

Vue Router

Axios

Bulma

Buefy

Element

Vditor

DarkReader
